Il valore assoluto ed il segno di un numero reale sono definiti:
$$
|x| = \left\{\begin{align}
x, &\text{ se } x \gt 0 \\
0, &\text{ se } x = 0 \\
x, &\text{ se } x \lt 0 \\
\end{align}
\right\|
$$
$$
sgn(x) = \left\{\begin{align}
1, &\text{ se } x \gt 0 \\
0, &\text{ se } x = 0 \\
-1, &\text{ se } x \lt 0 \\
\end{align}
\right\|
$$
